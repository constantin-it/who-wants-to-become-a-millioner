# Кто хочет стать миллионером? (Who Wants to Become a Millionaire?)

Это простая консольная игра-викторина, написанная на Python, вдохновленная популярным телешоу.

## Описание проекта

Проект представляет собой интерактивную викторину, где игроку предлагается ответить на серию вопросов. Каждый правильный ответ приближает игрока к главному призу — 1 000 000 рублей.

## Особенности

- **Тематика вопросов**: Музыка (Metallica, Motörhead, The Beatles) и аниме (Pokemon Horizons).
- **Механика**: 
  - Игроку предлагается 4 варианта ответа (A, B, C, D).
  - Один неверный ответ приводит к завершению игры.
  - Постепенное накопление виртуального выигрыша.
- **Простота**: Минималистичный интерфейс в командной строке.

## Как запустить

Для запуска игры вам понадобится установленный Python 3.

1. Склонируйте репозиторий или скачайте файл `who-wants-to-become-a-millioner.py`.
2. Откройте терминал в папке с файлом.
3. Запустите скрипт командой:
   ```bash
   python who-wants-to-become-a-millioner.py
   ```

## Пример игры

```text
--------------------------------------------------------
Добро пожаловать!
Вы на шоу 'Кто хочет стать миллионером?'
--------------------------------------------------------
Кто лидер 'Rising Volt Tacklers' в Pokemon Horizons?
A) Liko
B) Friede
C) Murdock
D) Dot
Ваш ответ: B
Правильный ответ! + 100000 рублей
--------------------------------------------------------
```
